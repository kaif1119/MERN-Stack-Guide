# 🤝 Contributing to Multer Mastery

**Shukriya ki aap contribute karna chahte hain!** Yeh repository community effort hai.

---

## 🚀 Kaise Contribute Karein?

### Step 1: Fork karo

```bash
# GitHub pe Fork button click karo
# Phir clone karo
git clone https://github.com/YOUR_USERNAME/multer-mastery.git
cd multer-mastery
```

### Step 2: Branch banao

```bash
# Feature ke liye:
git checkout -b feature/add-chapter-13

# Bug fix ke liye:
git checkout -b fix/typo-in-chapter-02

# Interview question add karne ke liye:
git checkout -b feat/add-interview-questions
```

### Step 3: Changes karo

- Apna content add karo
- Code examples test karo
- Hindi + English mix use karo

### Step 4: Commit karo

```bash
git add .
git commit -m "feat: Chapter 13 - WebSocket file uploads add kiya"
```

### Commit Message Format:

```
feat:  → Naya feature ya content
fix:   → Bug ya typo fix
docs:  → Documentation update
chore: → Maintenance work
```

### Step 5: Push aur PR banao

```bash
git push origin feature/add-chapter-13
# GitHub pe Pull Request open karo
```

---

## 📋 Contribution Guidelines

### ✅ Accept Ho Sakta Hai:

- Naye chapters ya topics
- Code examples improve karna
- Interview questions add karna
- Exercises add karna
- Typos fix karna
- Translations improve karna
- New projects add karna

### ❌ Accept Nahi Hoga:

- Plagiarized content
- Wrong ya untested code examples
- Off-topic content
- Spam PRs

---

## 📝 Content Format

Har chapter mein yeh structure follow karo:

```markdown
# Chapter XX: Topic Name

> Level: | Time: | Language:

## Table of Contents

## 1. Definition
## 2. Why It Exists
## 3. Real World Analogy
## 4. Syntax & Code (line by line explanation)
## 5. Best Practices
## 6. Common Mistakes
## 7. Interview Questions
## 8. Exercises
```

## 🐛 Bug Report

Issue create karo with:
- Kya problem hai
- Expected behavior kya tha
- Actual behavior kya hua
- Code snippet (agar applicable)
